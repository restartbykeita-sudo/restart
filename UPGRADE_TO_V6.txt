UPGRADE FROM V5 TO V6
=====================
1. Back up the website and database.
2. Upload all V6 website files.
3. Run supabase/006_no_email_confirmation_signup_hardening.sql.
4. In Supabase Dashboard disable Confirm email.
5. Clear site cache / unregister the old Service Worker once.
6. Logout and login again.
7. Test with a new email: it must appear in Authentication > Users and public.profiles as PENDING.
8. Admin approves the user in ผู้ใช้งานและสิทธิ์.

Fresh installation: run supabase/000_INSTALL_ALL_V6.sql.
